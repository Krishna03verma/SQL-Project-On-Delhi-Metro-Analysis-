## 📌 Overview

The Delhi Metro Network Analysis project leverages SQL to explore and analyze station-level and line-level data across multiple Delhi Metro routes. The project focuses on understanding network structure, station distribution, route connectivity, interchange stations, and metro line performance.

Using advanced SQL queries, the analysis transforms raw metro station data into actionable insights that help visualize the connectivity and operational structure of one of India's largest urban transportation systems.

---

## 🎯 Business Problem

Modern metro networks consist of multiple interconnected lines and stations. Analyzing these networks is essential for understanding route connectivity, passenger movement patterns, interchange dependencies, and network efficiency.

Without structured analysis, it becomes difficult to:

- Understand metro line coverage.
- Identify interchange stations.
- Analyze route connectivity.
- Evaluate network structure.
- Support transportation planning and decision-making.

---

## 🎯 Project Objective

The objective of this project is to analyze the Delhi Metro network using SQL and generate insights related to:

- Station distribution across metro lines.
- Junction and interchange stations.
- Metro route connectivity.
- Longest metro lines by station count.
- Station sequencing and route navigation.
- Transportation network analysis.

---

## 📊 Key Analysis Performed

### Station Count Analysis

- Calculated the total number of stations on each metro line.
- Compared station distribution across major metro routes.

### Junction Station Analysis

- Identified all junction stations across multiple metro lines.
- Analyzed key connectivity points within the network.

### Longest Metro Line Analysis

- Ranked metro lines based on total station count.
- Identified the top longest routes in the Delhi Metro network.

### Interchange Station Analysis

- Used SQL JOIN operations to identify common stations between metro lines.
- Analyzed connectivity between Blue, Yellow, Red, and Magenta lines.

### Route Navigation Analysis

- Applied Self Joins to determine previous and next stations for every station on a route.
- Created route sequencing logic for metro navigation analysis.

---

## 🔍 Key Insights

- Blue Line emerged as the longest metro line with the highest number of stations.
- Pink Line and Yellow Line are among the largest metro corridors in the network.
- Rajiv Chowk serves as a major interchange station connecting multiple lines.
- Kashmere Gate acts as one of the most important transportation hubs in the network.
- Junction stations play a critical role in improving route accessibility and passenger movement.
- Route sequencing analysis provides better visibility into station connectivity and travel paths.
